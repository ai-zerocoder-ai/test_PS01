Академия Кодебай

Онлайн-обучение по IT и информационной безопасности.

Программа: Paranoid 2.0

Период обучения: декабрь 2023 года - февраль 2024 года

Курсы и темы:

  • Инструменты шифрования и генерации паролей, парольных фраз и ключей
  • Полнодисковое шифрование с использованием cryptsetup (LUKS)
  • Продвинутая установка Linux (ядро, ключ шифрования и заголовок LUKS находятся в безопасном контейнере (/boot) на usb-носителе)
  • Восстановление через chroot
  • Обеспечение безопасности системы Linux (установка ядра linux-hardened, дополнительные настройки безопасности sysctl, защита от DMA и отключение модулей ядра, защита usb-портов (usbkill, usbguard, паническая кнопка, создание копий и восстановление хостовой системы)
  • Настройка сети (смена мак-адреса сетевого устройства, смена имени хоста, настройка системного файервола (opensnitch), работа с tor, tox, i2p сетями, настройка VPS и служб, обфускация сетевого трафика (OpenVPN + Stunnel), настройка генерации случайного https и dns трафика, разделителя ветвей сети)
  • Виртуализация (QEMU)
  • Работа с децентрализованными и федеративными мессенджерами (tox, jami, element)
  • Работа с криптоконтейнерами (veracrypt, zulucrypt), стеганография

Университет Зерокодинга

Курс: "Промпт-инжиниринг"

Дата завершения: сентябрь 2024 года

Основные модули и результаты обучения:

  1. Основы промпт-инжиниринга
    • Понимание роли промпт-инженера для бизнеса и актуальности этой профессии.
    • Изучение принципов работы больших языковых моделей (LLM) и их токенизации.
    • Разработка эффективных промптов и создание нейро-ассистентов для различных задач, от копирайтинга до диетологии.
    • Интеграция нейро-ассистентов со сторонними сервисами, такими как Google Calendar.
  2. API OpenAI
    • Практическое использование Playground OpenAI для тестирования моделей GPT и настройки запросов.
    • Автоматизация бизнес-задач с использованием GPT-API, таких как генерация текстов в Google-таблицах, транскрибация через Whisper и использование моделей Text-to-Speech (TtS).
    • Интеграция API для решения реальных бизнес-кейсов.
  3. Работа с базами данных
    • Основы работы с базами данных: сбор, хранение и очистка данных.
    • Проектирование эффективных баз данных для ИИ-проектов с учетом конфиденциальности данных.
    • Векторизация данных и использование их в LLM для улучшения качества ответов от нейросетей.
    • Практическая аналитика данных для бизнес-задач с использованием кода и нейросетей.
  4. Кейсы разработки ИИ-систем (текст)
    • Разработка ИИ-систем для работы с текстом, таких как генераторы постов для блогов, чат-боты для продаж, онлайн-консультанты и ИИ-ассистенты для прескоринга кандидатов.
    • Создание ИИ-систем для оптимизации бизнес-процессов и повышения эффективности работы команды.
  5. Кейсы разработки ИИ-систем (аудио, видео)
    • Разработка ИИ-систем для работы с аудио и видео, таких как транскрибация и перевод контента, а также системы распознавания речи для отдела продаж.
    • Оптимизация работы бизнеса за счет применения ИИ для аудио и видеоданных.
  6. Кейсы разработки ИИ-систем (изображения)
    • Разработка ИИ-систем для генерации текстового и визуального контента, таких как чат-боты с использованием GPT-Vision и инструменты для автоматизации маркетингового контента.
    • Использование Computer Vision и Stable Diffusion для создания визуального контента.
  7. Оптимизация затрат на ИИ. Стратегии развертывания ИИ-моделей
    • Сравнение зарубежных (Claude, LLaMA, Mistral AI) и российских (YandexGPT, GigaChat) моделей с открытым API в качестве альтернативы ChatGPT.
    • Оптимизация затрат на ИИ-проекты за счет выбора стратегий развертывания ИИ-моделей: локальных или облачных решений.
    • Понимание преимуществ и недостатков различных вариантов развертывания ИИ, включая масштабируемость, безопасность и стоимость.
  8. Основы работы с LangChain
    • Изучение фреймворка LangChain для создания интеллектуальных систем на основе больших языковых моделей.
    • Разработка автоматизированной HR-системы на базе LangChain и интеграция с Telegram для обработки типичных запросов сотрудников и кандидатов.
  9. Дообучение и разработка автономных агентов
    • Fine-tuning нейросетей для адаптации моделей под специфические задачи с использованием целевых данных.
    • Проектирование мультиагентных систем, состоящих из нескольких взаимодействующих ИИ-агентов, способных решать сложные задачи.
